Como surgiu os Computadores?

(Voltar)

Os computadores Surgiram em 1944, com a criacao do Mark 1,um computador eletromecanico gigantesco, para se ter uma base do seu tamanho,era necessario uma sala inteira para guarda-lo. Ele era um computador utilizado exclusivamente para fazer calculos e trabalhava números com 23 decimais, realizando as quatro operações aritméticas

Logo apos isso, em meados da Segunda Guerra Mundial(1946), na Alemanha, tivemos a criacao do computador chamado Colossus, tambem foi criado para fazer calculos e criptoanalises relacionados a Guerra. Do lado dos Estados Unidos, em paralelo ao Colossus, tivemos a criacao do Eniac, um computador automatico, perdendo aquela caracteristica eletromecanica do Mark 1. Foi nessa epoca que tivemos o surgimento do termo BUG(inseto), pois aparecia muitos insetos entre as partes mecanicas dos computadores, e quando um tecnico ia fazer a sua manutencao e se deparava com estes intsetos, eles utilizavam a expressao "deu bug".

Em 1959 tivemos a evolucao dos computadores para a segunda geracao, onde o tamanho dos computadores diminuiram consideravelmente de tamanho, sendo marcado pelos transistores, que substituiram as valvulas mecanicas.

A terceira geracao deu inicio em 1965, com o surgimento dos CI (circuito integrado), diminuindo ainda mais o tamanho dos computadores, porem, aumentando sua eficacia.

Em 1971, se deu inicio a quarta geracao, onde estamos ate os dias atuais, onde apareceu os computadores pessoais. Como exemplo, hoje temos acesso a computadores do tamanho de uma mao, os portateis cresceram ainda mais e a tecnologia so avanca a cada dia que passa.

Ainda podemos citar a quinta geracao, com a criacao dos ambientes virtuais, inteligencias artificiais, criptomoedas, entre algumas outras caracteristicas.

O que são Clients?

(Voltar)

O client é o usuario, quem esta buscando alguma informacao ou endereco na internet,é quem sempre inicia a comunicação com o servidor. Quando acessamos um site pelo navegador, uma requisição é disparada pela internet para o computador no qual esse site está hospedado.Após enviar uma requisição, o usuario aguarda até que haja uma resposta do servidor e então a comunicação se encerra.

O que são Servers?

(Voltar)

O papel do server, ou servidor, é receber uma requisição e devolver uma resposta para o cliente.O servidor é um ou mais computadores com um único endereço, nestes computadores podem ser hospedados documentos, imagens, vídeo, etc. Geralmente o servidor também hospeda programas que atendem as requisições dos clientes e as processam.

Estrutura básica de HTML

(Voltar)

Um documento HTML sempre tem seu inicio com uma estrutura basica, quase sempre imutavel.É possível compreender o documento em HTML de uma maneira muito simples, através de uma divisão de blocos das tags essenciais, conforme a a seguinte estrutura:

  • Definição do documento (doctype)
  • Cabeça (head)
  • Corpo (body)

O Doctype não é uma tag HTML, mas uma instrução para o navegador e outros programas que podem ler seu site, que o código encontrado ali é um código HTML. Assim eles sabem o que fazer para mostrar seu site da melhor forma possível.

A Cabeça(head) contem informações que não são transpostas visivelmente para o usuário/leitor do documento. São dados implícitos, de uso e controle do documento: vinculação com outros arquivos, aplicação de lógica de programação de scripts e metadados.

O Corpo(body)trata-se do documento em si, ou seja, a informação legível para o usuário/leitor do documento.

Listas Ordenadas e nao Ordenadas

(Voltar)

A principal diferença entre a lista ordenada e a lista não-ordenada está na maneira com que os itens de cada lista são apresentados - a lista ordenada enumera os elementos; a lista não-ordenada coloca marcadores antes de cada elemento.

Uma lista ordenada é delimitada pelas etiquetas OL (Ordered List). Cada item da lista é delimitado pelas etiquetas LI (List Item).

A estrutura da lista nao-ordenada é semelhante a da lista ordenada - porém, ao invés de ser construída com o elemento OL, é construída com o elemento UL (Unordered List).
